# KnowYourVote

[Demo site available](https://know-your-vote.vercel.app/)

## Overview

KnowYourVote is a web app designed to help citizens further understand their civic knowledge in the form of lessons and quizzes, while having access to a directory of congressional members along with the bills they sponsor

## Technical Architecture 
* **Next.js and Vercel:** Used to serve the frontend interface while also leveraging Server Components and Actions
* **Flask and LangGraph:** Used to serve the main RESTful API for the placement assesment
* **DigitalOcean App Platform:** Used to serve the flask repository files
